# aspipe_v4 数据平台 - 第一期实现

## 项目概述

第一期实现了单文件完整数据下载系统，能够下载沪深A股的基本信息和日线数据并保存为Parquet格式。

## 核心功能

### 1. TuShare API集成
- ✅ 从.env文件读取token
- ✅ API限频控制
- ✅ 重试机制

### 2. 数据下载
- **股票基本信息**：获取所有上市股票的基本信息（代码、名称、行业、地域等）
- **日线数据**：下载指定时间范围的后复权日线数据
- **限制**：第一期限制下载前50只股票以避免超时

### 3. 数据存储
- ✅ 使用Parquet格式存储
- ✅ 股票基本信息：`data/stock_basic.parquet`
- ✅ 日线数据：`data/daily_hfq.parquet`

### 4. 错误处理与验证
- ✅ 网络异常处理
- ✅ 数据完整性验证
- ✅ 价格逻辑验证

### 5. 进度跟踪
- ✅ 详细的中文日志输出
- ✅ 实时进度显示
- ✅ 结果统计

## 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 运行系统
python main.py
```

