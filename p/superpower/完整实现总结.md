## 技术架构

### 1. SchemaManager - 派生字段核心
```python
# 派生字段应用在 SchemaManager.create_dataframe()
df = SchemaManager.apply_derived_fields(df, interface_name)
```

### 2. DataProcessor - 数据流水线控制
- 确保主键空值过滤发生在派生字段之后
- 防止对空值进行不必要的转换操作

### 3. DateConverter - 高性能转换器
- 三级缓存策略：内存缓存 → 本地存储 → API请求
- 支持惰性加载，避免启动性能开销

### 4. 配置驱动 - 统一管理
- 所有接口在 `app4/config/interfaces/` 目录下
- 派生字段通过 `derived_fields` 统一配置

## 使用示例

### Polars查询优化
```python
import polars as pl
from datetime import date

# 读取数据
df = pl.read_parquet("app4/data/daily/*.parquet")

# 使用日期格式进行高效过滤（性能提升3-5倍）
result = df.filter(
    pl.col("trade_date_dt").is_between(date(2024, 1, 1), date(2024, 12, 31))
)

# 日期运算性能更好
result = df.with_columns(
    (pl.col("trade_date_dt") + pl.duration(days=1)).alias("next_day")
)

# 与trade_cal关联（无需类型转换）
trade_cal = pl.read_parquet("app4/data/trade_cal/*.parquet")

result = daily.join(
    trade_cal.select(["cal_date_dt", "is_open_bool"]),
    left_on="trade_date_dt",
    right_on="cal_date_dt",
    how="left"
)
```
