### 2.1 修复日期参数问题

**问题**: 使用未来日期（20251231）导致大部分接口无数据返回

**修复方案**:
```python
from datetime import datetime

def validate_and_adjust_date(start_date, end_date):
    """
    验证并调整日期，确保不使用未来日期
    """
    today = datetime.now().strftime('%Y%m%d')
    
    if end_date > today:
        print(f"警告: 结束日期 {end_date} 超过当前日期，已调整为 {today}")
        end_date = today
    
    if start_date > today:
        print(f"警告: 开始日期 {start_date} 超过当前日期，已调整为 {today}")
        start_date = today
        
    return start_date, end_date
```

**修改位置**: `/home/quan/testdata/aspipe_v4/app4/main.py`

### 2.2 修复broker_recommend接口参数问题

**问题**: broker_recommend接口需要month参数（格式：YYYYMM），但系统传递的是start_date和end_date

**修复方案**:
```python
def handle_broker_recommend(ts_code, start_date, end_date, **kwargs):
    """
    broker_recommend接口需要month参数，将日期范围转换为月份列表
    """
    import pandas as pd
    from datetime import datetime

    # 将日期范围转换为月份列表
    start = pd.to_datetime(start_date)
    end = pd.to_datetime(end_date)

    # 生成月份列表
    months = pd.date_range(start=start, end=end, freq='M').strftime('%Y%m').tolist()

    all_data = []
    for month in months:
        try:
            data = pro.broker_recommend(month=month)
            all_data.append(data)
        except Exception as e:
            print(f"Error fetching broker_recommend for month {month}: {e}")
            continue

    return pd.concat(all_data, ignore_index=True) if all_data else pd.DataFrame()
```

**修改位置**: `/home/quan/testdata/aspipe_v4/app4/main.py`

### 2.4 简化去重逻辑

**问题**: 多个接口存在重复数据

**修复方案**:
```python
def simple_dedup(data, key_columns=None):
    """
    简单去重函数
    """
    if data.empty:
        return data

    # 如果没有指定关键列，使用默认的唯一标识
    if key_columns is None:
        if 'ts_code' in data.columns and 'trade_date' in data.columns:
            key_columns = ['ts_code', 'trade_date']
        elif 'ts_code' in data.columns:
            key_columns = ['ts_code']
        else:
            # 如果没有明显的关键列，使用所有列
            key_columns = data.columns.tolist()

    # 按关键列去重，保留最后一条记录（最新数据）
    return data.drop_duplicates(subset=key_columns, keep='last').reset_index(drop=True)
```
