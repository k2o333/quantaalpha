## 接口1: cyq_chips（每日筹码分布）

### 1.1 问题描述

**日志证据**:
```
2026-01-02 10:15:54,282 - core.downloader - WARNING - Window 20250630-20251231 returned 6000 records, which may be truncated (API limit: 2000)
2026-01-02 10:15:54,292 - core.downloader - WARNING - Found 5943 duplicate records for 000001.SZ, removed from 6000 to 57 records
```

**问题分析**:
- API限制：单次最大2000条记录
- 实际返回：6000条记录（被截断）
- 去重后：仅57条有效记录
- 日期窗口：20250630-20251231（未来日期，无实际数据）

```yaml
    type: string
```

### 1.3 修复方案

**方案A：减小日期窗口大小（推荐）**

修改配置文件 `/home/quan/testdata/aspipe_v4/app4/config/interfaces/cyq_chips.yaml`：

```yaml
pagination:
  enabled: true
  mode: "date_range"
  window_size_days: 30  # 从365改为30，确保单次请求不超过2000条
```

**修改说明**:
- 原窗口365天，对于单个股票可能返回大量数据（每个交易日可能有多个价格点）
- 改为30天窗口，确保单次请求不超过2000条限制
- 系统会自动按30天窗口分批请求，避免数据截断

**方案B：按股票代码分批请求**

如果方案A仍然无法解决问题，可以在downloader中添加特殊处理：

在 `/home/quan/testdata/aspipe_v4/app4/core/downloader.py` 中添加：

```python
def _execute_cyq_chips_special(self, interface_config: Dict[str, Any], params: Dict[str, Any]) -> List[Dict[str, Any]]:
    """
    cyq_chips接口特殊处理：按日期分批请求，确保不超过2000条限制
    """
    from datetime import datetime, timedelta

    start_date = params.get('start_date')
    end_date = params.get('end_date')
    ts_code = params.get('ts_code')

    if not start_date or not end_date:
        return self._make_request(interface_config, params)

    # 将日期范围拆分为30天的小窗口
    start_dt = datetime.strptime(start_date, '%Y%m%d')
    end_dt = datetime.strptime(end_date, '%Y%m%d')

    all_data = []
    current_dt = start_dt

    while current_dt <= end_dt:
        window_end_dt = min(current_dt + timedelta(days=30), end_dt)
        window_start = current_dt.strftime('%Y%m%d')
        window_end = window_end_dt.strftime('%Y%m%d')

        window_params = params.copy()
        window_params['start_date'] = window_start
        window_params['end_date'] = window_end

        window_data = self._make_request(interface_config, window_params)
        if window_data:
            all_data.extend(window_data)

        current_dt = window_end_dt + timedelta(days=1)

    return all_data
```

**修改位置**: 在第457行（普通下载逻辑）之前添加

```python
# 特殊接口处理：broker_recommend需要month参数
if interface_name == 'broker_recommend':
    import pandas as pd
    from datetime import datetime

    # 转换日期范围为月份列表
    start_dt = datetime.strptime(params['start_date'], '%Y%m%d')
    end_dt = datetime.strptime(params['end_date'], '%Y%m%d')

    # 生成月份列表
    months = pd.date_range(
        start_dt,
        end_dt,
        freq='MS'  # 月初
    ).strftime('%Y%m').tolist()

    all_data = []
    for month in months:
        month_params = {'month': month}

        # 如果有ts_code参数，也传递
        if 'ts_code' in params:
            month_params['ts_code'] = params['ts_code']

        # 下载该月数据
        interface_rate_limit = interface_config.get('permissions', {}).get('rate_limit', global_rate_limit)
        interface_rate_limiter = RateLimiter(interface_rate_limit)
        interface_rate_limiter.wait_for_tokens(1)

        month_data = downloader.download(interface_name, month_params)

        if month_data:
            all_data.extend(month_data)
            logger.info(f"Downloaded {len(month_data)} records for {month}")

    if all_data:
        logger.info(f"Successfully downloaded {len(all_data)} total records for {interface_name}")
        process_and_save_data(all_data, interface_name, interface_config, processor, storage_manager)
    else:
        logger.warning(f"No data downloaded for {interface_name}")

    continue  # 跳过普通下载逻辑
```

```python
def validate_and_adjust_date(start_date, end_date):
    """
    验证并调整日期，确保不使用未来日期
    """
    from datetime import datetime

    today = datetime.now().strftime('%Y%m%d')

    if end_date > today:
        logger.warning(f"结束日期 {end_date} 超过当前日期，已调整为 {today}")
        end_date = today

    if start_date > today:
        logger.warning(f"开始日期 {start_date} 超过当前日期，已调整为 {today}")
        start_date = today

    return start_date, end_date
```
