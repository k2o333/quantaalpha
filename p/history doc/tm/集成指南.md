# StockListManager 集成指南

## 快速集成步骤

### 第一步：创建核心文件

在 `/home/quan/testdata/aspipe_v4/app/` 目录下创建两个文件：
1. `stock_list_manager.py` - StockListManager 核心实现
2. （可选）`cache_manager.py` - 缓存管理工具

### 第二步：修改程序入口

在 `enhanced_main_downloader.py` 或 `main.py` 中添加初始化代码：

```python
from stock_list_manager import init_stock_manager

# 在程序启动时初始化（在创建 downloader 之后）
stock_manager = init_stock_manager(
    downloader=tushare_downloader,
    cache_dir="cache",
    max_cache_age_hours=24
)
```

### 第三步：替换现有调用

将所有 `download_stock_basic()` 调用替换为 StockListManager 调用：

**原代码：**
```python
stock_df = self.downloader.download_stock_basic()
```

**新代码：**
```python
from stock_list_manager import StockListManager

stock_df = StockListManager().get_stock_basic()
```

## 常见问题解决

### Q1: 缓存文件损坏怎么办？
A: StockListManager 会自动检测并重新下载，也可以手动调用 `clear_cache()` 清除缓存。

### Q2: 多线程环境下的安全性？
A: 实现时添加了线程安全机制，但在高并发场景下建议添加锁机制。

### Q3: 内存占用问题？
A: StockListManager 使用单例模式，内存占用很小。如果数据量很大，可以考虑惰性加载。

### Q4: 如何强制刷新数据？
A: 调用 `get_stock_basic(force_refresh=True)` 或 `refresh_cache()` 方法。
